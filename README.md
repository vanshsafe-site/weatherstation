# 🌦️ Weather Station

> **Professional Weather Intelligence for Everyone**

A modern, cockpit-inspired weather application that transforms weather forecasting into an immersive experience. Weather Station provides real-time weather updates, interactive maps, air quality monitoring, storm intelligence, and detailed meteorological insights through a beautiful, responsive, and high-performance interface.

---

## ✨ Features

- 🌡️ Real-time weather conditions
- 📅 Hourly, daily, and weekly forecasts
- 🌍 Interactive weather maps
- 🌩️ Storm tracking and weather alerts
- 🌬️ Wind, humidity, pressure, and visibility analysis
- 🌫️ Air Quality Index (AQI) monitoring
- ☀️ UV Index and sunrise/sunset information
- 📊 Interactive weather charts
- 📱 Fully responsive design
- 🌙 Dynamic weather-based themes
- ⚡ Fast loading and optimized performance
- 💾 Progressive Web App (PWA) support
- 🔒 Privacy-focused with secure location access

---

## 🖥️ Built With

- React
- TypeScript
- Vite
- Tailwind CSS
- OpenWeather API
- Open-Meteo API
- WeatherAPI

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/vanshsafe-site/Weatherwithvansh.git
```

### Navigate to the project

```bash
cd Weatherwithvansh
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm run dev
```

Open your browser and visit:

```
http://localhost:5173
```
